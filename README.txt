# WHAT IF? Web Page

## Overview
This project is a simple web page created for RM Gayler to showcase their work, share updates, and provide contact information.
The site includes information about their books, upcoming events, and a blog.

## Features
- Homepage with an introduction to the author
- List of published books with descriptions
- Contact form for readers to get in touch

## Technologies Used
- HTML for the structure
- CSS for styling
- JavaScript for interactive features
